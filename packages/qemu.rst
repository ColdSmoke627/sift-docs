qemu
====
QEMU is a generic and open source machine emulator and virtualizer. 

* **Website:** http://wiki.qemu.org/Main_Page

Raw Image to VMDK
^^^^^^^^^^^^^^^^^
.. code-block:: html

    $ qemu-img convert imagefile.dd -O vmdk vmdkname.vmdk
